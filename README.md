# anonymous_data

Kleine Demo-Webseite, die in einem Eingabefeld alle Ziffern (0–9) durch `*` ersetzt.

Dateien
- index.html — Hauptseite: Eingabe- und Ausgabefeld (Ziffern werden durch `*` ersetzt)

Verwendung
- Lokales Öffnen: Doppelklick auf `index.html` oder im Browser `Datei öffnen`.
- Lokaler Webserver (empfohlen):

  ```bash
  python -m http.server 8000
  # dann öffnen: http://localhost:8000/index.html
  ```

Kurzer Hinweis
- Die Ersetzung läuft clientseitig (JavaScript). Keine zusätzlichen Abhängigkeiten.
